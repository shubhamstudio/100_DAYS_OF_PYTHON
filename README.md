# 100 Days of Python Challenge 🐍

Welcome to my personal repository tracking my journey through the **100 Days of Code: The Complete Python Pro Bootcamp**. This repository contains all my daily projects, challenges, and source code organized by course sections.

## 🚀 Getting Started

### Prerequisites
Make sure you have Python 3.10+ installed on your local machine.

### Installation & Local Setup
1. Clone this repository:
   ```bash
   git clone https://github.com
   ```
2. Navigate to the project root directory:
   ```bash
   cd 100_DAYS_OF_PYTHON
   ```
3. Run any specific project script (e.g., inside PART1):
   ```bash
   python PART1/path_to_script.py
   ```

## 🛠️ Tech Stack & Tools
* **Language:** Python 3
* **Environment:** Git Bash / MINGW64 (Windows)
* **IDE:** VS Code / PyCharm
